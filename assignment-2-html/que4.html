This is a two types list ordered list and unordered list. Each list contained list tags "li"

  1. h tag is used for heading
  2. p tag is used for paragraph
  3. div tag is used as a container
  4. a tag is used for hyperlinks
  5. li tag is used for list
  6. i tag is used for italic
  7. b tag is used for bold
  8. u tag is used for underline
