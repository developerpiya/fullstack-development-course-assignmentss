Properties of flexbox

justify-content

justify-content set the flex items horizontly ex- set the flex item center,flex-start,flex-end,space-between, space-around

justify-content set the flex items vertically ex- set the flex item center,flex-start,flex-end,space-between, space-around

gap is used for space between the flex items

flex-direction is used to define the direction like- column,row,row-reverse, column-reverse

flex-wrap is used to wrap the flex items
