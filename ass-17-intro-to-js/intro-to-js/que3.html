Que3 - List and Descibe the five primitives data types in js ?

Ans - Primitives Data Types-------

  • StringA string is a collection of alphanumeric collection. It's written withing single, double quotes and backtick character ex:- let color = "Yellow";
  • Number - Number is used in numbers whereas it is integer or decimal ex:- let length = 16;
  • Boolean- Basically boolean is used only for true/false or 0/1 ex:- let x = true;
  • undefined - In JavaScript, a variable without a value, has the value undefined. ex :- let car = undefined
  • BigInt - All JavaScript numbers are stored in a a 64-bit floating-point format.It can be store the values that are too big ex:- let x = BigInt("123456789012345678901234567890");
